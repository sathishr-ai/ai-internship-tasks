## 📖 Comprehensive Technical Documentation

### 🚀 Introduction
This repository contains a collection of four distinct AI applications developed during my intensive 4-week internship at **CODTECH IT SOLUTIONS**. The goal was to master the implementation of state-of-the-art AI models across different domains, including Natural Language Processing, Computer Vision, and Speech Recognition.

### 🔬 Technical Deep Dive

#### 💡 Task 1: NLP Text Summarization
The challenge was to automate the extraction of key information from large texts. I implemented a solution using the **Transformer (T5)** architecture. This model treats every NLP problem as a "text-to-text" task, allowing for highly accurate summarization that maintains semantic integrity while reducing length by up to 80%.

#### 💡 Task 2: Speech Recognition System
I developed a functional speech-to-text system optimized for accessibility. By leveraging the `SpeechRecognition` library, the tool can process audio inputs and convert them into digital text. This project explores the challenges of acoustic modeling and linguistic decoding in real-time environments.

#### 💡 Task 3: Neural Style Transfer
Using the **VGG19** model, I implemented a style transfer pipeline that utilizes **Feature Inversion** and **Gram Matrices**. By extracting content from deeper layers and style from multiple shallow layers, the model creates a unique artistic fusion, demonstrating a deep understanding of convolutional neural network feature spaces.

#### 💡 Task 4: Generative Text Modeling
The final project involved fine-tuning a **GPT-2** model for text generation. By adjusting hyper-parameters such as Top-K and Top-P, I ensured the model generates text that is neither too repetitive nor overly random. It produces high-quality, coherent paragraphs suitable for a wide range of creative applications.
