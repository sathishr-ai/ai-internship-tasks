## 📖 Detailed Task Descriptions

### 1. NLP Text Summarization (Task 1)
Created a state-of-the-art summarization tool using the **T5 (Text-to-Text Transfer Transformer)** model. This task addresses the modern challenge of information density by providing a system that can process lengthy documents and return high-fidelity summaries. By utilizing the Hugging Face `Transformers` library, the script achieves a high BLEU score and ensures that the core context of the source material is preserved in a significantly compressed format.

### 2. Speech-to-Text Recognition (Task 2)
Developed a robust speech transcription engine using the `SpeechRecognition` library. Optimized for local execution on Windows environments, this tool processes WAV audio and interfaces with cloud-based recognition APIs to provide near-instantaneous text output. This implementation demonstrates proficiency in handling multi-modal data and asynchronous API communications.

### 3. Neural Style Transfer (Task 3)
Implemented an advanced computer vision model based on the **VGG19** deep learning architecture. This task leverages **Transfer Learning** to separate the "content" and "style" features of two distinct images. Using a custom optimization loop with TensorFlow, the model iteratively minimizes content and style loss functions to produce an original piece of digital art that blends photographic reality with painterly aesthetics.

### 4. Generative AI Text Model (Task 4)
Built a generative text engine powered by **GPT-2**, capable of autocompleting and extending user prompts into coherent, multi-paragraph narratives. The implementation features advanced sampling techniques like *Top-K* and *Top-P (Nucleus Sampling)* to ensure the variety and natural flow of the generated text, making it suitable for creative writing assistants and automated content generation.

## 🛠️ Installation & Usage

### 📦 Prerequisites
- Python 3.8+
- Active Internet Connection (for model downloads)

### 🚀 Setup
```powershell
# Clone the repository
git clone https://github.com/sathishr-ai/ai-internship-tasks.git

# Install dependencies
pip install -r requirements.txt
```

### 💻 Run Commands
| Task | PowerShell Command |
| :--- | :--- |
| **Summarizer** | `python task1_summarization/text_summarizer.py` |
| **Speech-to-Text** | `python task2_speech/speech_to_text.py` |
| **Style Transfer** | `python task3_style_transfer/style_transfer.py` |
| **Text Gen** | `python task4_text_gen/text_generator.py "Your Prompt Here"` |
